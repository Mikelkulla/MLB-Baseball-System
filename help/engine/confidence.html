Confidence & Tier Assignment

The V8.0 two-component confidence formula, SharpSplit, WPI, tier gating, SafeUnits rules, and safety caps.

The Two Components

Confidence is an equal-weighted blend of EV strength and sharp-money signal.

Component 1 — EV Normalised (evNorm)

evNorm — EV scaled to 0–80
evNorm = MIN(80, MAX(0, 50 + ev_pct × 2))

# ev_pct = None (outside gate) → evNorm = 50.0 (neutral)
# ev_pct = 0% → evNorm = 50
# ev_pct = +5% → evNorm = 60
# ev_pct = +15% → evNorm = 80 (capped)

Component 2 — Sharp Score

Two sub-values feed SharpScore:

SharpSplit — signed handle vs bets gap
sharp_split = ourHandle% − ourBets%

# Positive: sharp money backing our pick (handle > bets = fewer, larger bets)
# Negative: public money dominates (bets > handle = many small bets)
# ML bets% and handle% are EXCLUDED (only spread + totals per V8.0)
WPI — Whale Pressure Index
WPI = MIN(100, MAX(0, 50 + (ourHandle% − 50) × 1.5))

# ourHandle% = 50% → WPI = 50 (neutral / no data)
# ourHandle% = 70% → WPI = 80 (strong sharp alignment)
# ourHandle% = 30% → WPI = 20 (sharp money against us)
SharpScore
SharpScore = MIN(100, MAX(0, 50 + sharp_split × 0.5 + (WPI − 50) × 0.4))

Final Confidence

V8.0 Confidence Formula (range: 25–95)
Confidence = MIN(95, MAX(25, evNorm × 0.50 + SharpScore × 0.50))

Tier Assignment

Confidence alone doesn't assign a tier — WPI must also meet a minimum threshold. A high-confidence pick backed by weak WPI gets downgraded.

TierMin ConfidenceMin WPIUnits
ELITE ≥ 85%≥ 753.0u
STRONGEST ≥ 75%≥ 652.5u
BEST BET ≥ 68%≥ 551.75u
GOLD ≥ 55%≥ 01.0u
PASS < 55%any0u

If confidence qualifies for ELITE (≥85%) but WPI is only 60 (below the 75 threshold), the tier downgrades to STRONGEST, then checks WPI ≥ 65, and so on until a tier's WPI gate is met or PASS is assigned.

Safety Layers

Four safety mechanisms can reduce or zero out the recommended units after tier assignment. The tier label is preserved (so you can still see the model's view), but safe_units reflects the risk-adjusted position size.

1. Steam-Against Cap (≥1.5 pts CLV adverse)

Steam-Against
if clv_delta ≤ −1.5:
    confidence = MIN(74%, confidence) # cap at 74%

2. Steam Auto-Pass (≥2.0 pts CLV adverse)

Steam Auto-Pass
if clv_delta ≤ −2.0:
    safe_units = 0 # keep tier, zero units

3. LineFlip Cap

LineFlip
if spread sign changed AND SharpScore < 70:
    confidence = MIN(74%, confidence) # cap at 74%

4. SP Gate

SP Gate
if sp_gate_blocked = True:
    safe_units = 0 # probable SP is Out/Doubtful

SafeUnits Rules (applied in order)

PriorityConditionResult
1SP gate blockedsafe_units = 0
2WPI < 50safe_units = 0, status = "ACTION ONLY"
3Steam/LineFlip triggeredsafe_units = min(base_units, 1.0u)
4WPI < 60 AND base units ≥ 2.5usafe_units = 1.75u (downgrade)
5No rule firedsafe_units = base_units (keep tier)
ACTION ONLY

When WPI < 50 (sharp money against us), the pick is shown with 0 units but labelled "ACTION ONLY" rather than PASS. This means the model sees EV value in the pick but sharp-money alignment is too weak to recommend a bet size. It's a watch signal, not a qualified pick.
