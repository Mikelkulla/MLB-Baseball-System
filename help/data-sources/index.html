All five external feeds — what they provide, how they're fetched, and where the API keys live.

The Odds API

FieldValue
Filedata/odds_client.py
API key settingconfig/settings.py → ODDS_API_KEY
Refresh interval30 min (odds job) + 360 min (full refresh)
Markets fetchedh2h (moneyline), spreads, totals

Returns a Game object for every scheduled MLB game today, with:

  • Away/Home moneyline odds
  • Away/Home run line (always ±1.5) and price
  • O/U total line and price
  • Opening spread stored in GameOdds.away_spread_open / home_spread_open (used for CLV calculation)
  • Multi-book consensus probability and best-price line across all available bookmakers
# Base URL
https://api.the-odds-api.com/v4/sports/baseball_mlb/odds/
?apiKey=...®ions=us&markets=h2h,spreads,totals&oddsFormat=american

DraftKings Network — Betting Splits

FieldValue
Filedata/draftking_scraper.py
MethodPlain HTTP GET — no Playwright needed
Refresh interval180 min
Sport ID84240 (MLB)

Returns handle% and bets% for every available market (ML, spread, totals) per game. These feed SharpSplit and WPI calculations.

# URL template
https://dknetwork.draftkings.com/draftkings-sportsbook-betting-splits/
?tb_eg=84240&tb_edate=n7days&tb_emt=0

# HTML parsing: split by <div class="tb-se"> per game
# Handle% is 3rd <div>, Bets% is 4th <div> in each option block
Old sportsbook URL gives 403

The sportsbook.draftkings.com URL returns 403 errors. Use only the DK Network URL above. The Network page is server-side rendered so plain requests + BeautifulSoup is sufficient.

SharpSplit (SSS)

SSS uses the maximum gap between bets% and handle% across spread and totals markets. Moneyline is excluded per V8.0 spec.

SSS = max(
    |away_spread_bets − away_spread_handle|,
    |home_spread_bets − home_spread_handle|,
    |over_bets − over_handle|,
    |under_bets − under_handle|,
)

WeatherAPI.com

FieldValue
Filedata/weather_client.py
API key settingconfig/settings.py → WEATHER_API_KEY
Refresh interval240 min

Fetches current weather for each stadium city. Dome stadiums skip weather adjustments automatically. Returns temperature (°F), wind speed (mph), wind direction (degrees from N), and precipitation category.

# Endpoint

Covers.com — Injury Scraper

FieldValue
Filedata/injury_scraper.py
MethodHTTP scrape (BeautifulSoup)
Refresh interval120 min
URLhttps://www.covers.com/sport/baseball/mlb/injuries

Returns RawInjury objects: player name, team, position, and status. Position is mapped to the weight table in mlb_config.py.

SP Gate trigger: if a player's position = SP and status is Out/Doubtful, sets sp_gate_blocked = True for that team's game.

Spring training stats are empty

The MLB Stats API returns no 2026 season stats until regular season starts in April. During spring training, all pitcher scores default to 50 (TBD) and all bullpen scores default to 50 (fewer than 10 games). No probability adjustments fire. This is correct and expected.
