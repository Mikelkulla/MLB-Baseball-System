Formula Reference

Every formula in the system — one page, quick lookup.

1. Vig Removal

raw_prob (odds ≥ 0) = 100 / (odds + 100)
raw_prob (odds < 0) = |odds| / (|odds| + 100)
total = raw_away + raw_home
true_away = raw_away / total × 100
true_home = raw_home / total × 100

2. Injury Probability Adjustment

net = home_injury_delta − away_injury_delta
new_away = away_prob + net / 2
new_home = home_prob − net / 2
→ re-normalise to sum = 100
Per-player delta
delta = position_weight × status_multiplier
team_impact = Σ(delta × diminishing_factor) capped at 12.0

3. Pitcher Probability Adjustment

edge = away_pitcher_score − home_pitcher_score
prob_shift = (edge / 50) × 5.0 × park_scaling # max ±10pp
new_away = away_prob + prob_shift
new_home = home_prob − prob_shift
→ re-normalise to sum = 100

4. Bullpen Probability Adjustment

edge = away_bullpen_score − home_bullpen_score
prob_shift = (edge / 50) × 2.0 × park_scaling # max ±4pp
new_away = away_prob + prob_shift
→ re-normalise to sum = 100

5. Park Pitcher Scaling

park_scaling = CLAMP(2.0 − park_factor, 0.60, 1.25)

6. EV Calculation

decimal (odds ≥ 0) = 1 + odds / 100
decimal (odds < 0) = 1 + 100 / |odds|
EV% = ((prob/100 × (decimal − 1)) − ((1 − prob/100) × 1)) × 100
# Returns None if |odds| > 400

7. Pick Selection

pick = argmax(away_ev, home_ev)
if best_ev < 0: → PASS

8. SharpSplit & WPI

sharp_split = ourHandle% − ourBets% # ML excluded; spread + totals only
WPI = CLAMP(50 + (ourHandle% − 50) × 1.5, 0, 100)

9. Confidence Formula

evNorm = CLAMP(50 + ev_pct × 2, 0, 80)
SharpScore = CLAMP(50 + sharp_split × 0.5 + (WPI − 50) × 0.4, 0, 100)
Confidence = CLAMP(evNorm × 0.50 + SharpScore × 0.50, 25, 95)

11. SafeUnits Rules

1. SP gate blocked → safe_units = 0
2. WPI < 50 → safe_units = 0, "ACTION ONLY"
3. Steam/LineFlip triggered → safe_units = min(units, 1.0u)
4. WPI < 60 AND units ≥ 2.5u → safe_units = 1.75u
5. No rule → safe_units = tier_units

12. Safety Caps

Steam-against (CLV ≤ −1.5 pts): confidence = min(74%, confidence)
Steam auto-pass (CLV ≤ −2.0 pts): safe_units = 0
LineFlip + SharpScore < 70: confidence = min(74%, confidence)

13. Pitcher Score (0–100)

FIP = ((13×HR) + (3×(BB+HBP)) − (2×K)) / IP + 3.17

fip_score = (8.40 − FIP) / 8.40 × 100 # league_avg=4.20; 2×avg=8.40
k9_score = K/9 / 17.60 × 100 # league_avg=8.80; 2×avg=17.60
bb9_score = (6.20 − BB/9) / 6.20 × 100 # league_avg=3.10; 2×avg=6.20
hr9_score = (2.30 − HR/9) / 2.30 × 100 # league_avg=1.15; 2×avg=2.30

score = fip_score×0.40 + k9_score×0.25 + bb9_score×0.20 + hr9_score×0.10 + recent_era_score×0.05

14. Bullpen Score (0–100)

score = fip_score×0.45 + k9_score×0.30 + bb9_score×0.25
# Same normalisation as pitcher score; games < 10 → return 50.0

15. Park O/U Adjustment

park_ou_adj = (park_factor − 1.0) × 5.3
# Colorado Rockies excluded (altitude handled by weather engine)
