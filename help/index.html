Quick Start

# Start the server
uvicorn app:app --reload --port 8000

# Open the dashboard
http://localhost:8000

# Live Picks (qualified bets only)
http://localhost:8000/predictions

# MLB Model (all games including PASS)
http://localhost:8000/model

# Trigger a manual full refresh
POST http://localhost:8000/api/predictions/refresh
Spring Training Note (March)

MLB Stats API returns no 2026 season stats until regular season starts in April. All pitcher and bullpen scores default to 50 (neutral) — no probability adjustment fires. This is correct behavior: the system won't generate qualified picks until real stats are available.
